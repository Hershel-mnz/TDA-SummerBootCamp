# Prompt Engineering Playground

## Overview

This project demonstrates Prompt Engineering using Google's Gemini AI model.

Users can enter text, select different prompt modes, and observe how the same input generates different outputs depending on the prompt used.

## Features

* Explain Like I'm 5
* Professional Rewrite
* Summarize Text
* Quiz Generator
* Mentor Mode
* Empty Input Validation
* Loading Animation

## Technologies Used

* Python
* Streamlit
* Google Gemini API
* python-dotenv

## Installation

Install required packages:

pip install -r requirements.txt

## Setup

Create a .env file:

GEMINI_API_KEY=YOUR_API_KEY

## Run

streamlit run app.py

## Example

Input:
What is Machine Learning?

Mode:
Explain Like I'm 5

Output:
A simple child-friendly explanation generated by Gemini.

## Learning Outcome

This project demonstrates how changing prompts can significantly alter the quality, style, and structure of responses generated by an LLM.
