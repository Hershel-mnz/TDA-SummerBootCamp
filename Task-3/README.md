## Project Description

AI Image Generator is a Streamlit-based web application that creates AI-generated images from user-provided text prompts. Users can enter a description of the image they want, select a preferred artistic style such as Anime, Cyberpunk, Fantasy, or Realistic, and generate an image using the Hugging Face Inference API.

The application demonstrates the integration of Generative AI, prompt engineering, and API-based image generation within an interactive web interface. It also maintains a prompt history, allowing users to review previously generated prompts. API keys are securely managed using environment variables, making the application suitable for local development and cloud deployment.

This project was developed as part of the TDA GenAI & Agentic AI Bootcamp Week 3 assignment to provide hands-on experience with image generation models, Streamlit development, API integration, and AI application deployment.
